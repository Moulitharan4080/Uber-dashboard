Uber Multi-Category Analysis Dashboard

Project Overview

This project is an end-to-end data analysis and visualization of Uber ride data covering 7 ride categories:

Auto

Bike

eBike

Go Mini

Go Sedan

Premier Sedan

Uber XL

The data was downloaded in Excel, cleaned and modeled in Power BI, and used to build an interactive dashboard showing business performance across the year.
